# 🚗 Car Parser & Listing (ENCAR)

Простое веб-приложение на FastAPI для парсинга автомобилей с ENCAR и отображения их в виде современного каталога.

## 🔧 Функционал

### 1. Парсинг ENCAR
- Получение данных через API:
  - марка (brand)
  - модель (model)
  - год (year)
  - пробег (mileage)
  - цена (price)
  - изображения (images)
- Сохранение данных в JSON (`data/cars.json`)
- Автоматическое обновление данных (scheduler)

### 2. Веб-интерфейс (Landing Page)
- Список автомобилей в виде карточек
- Карусель изображений
- Отображение:
  - названия
  - года
  - пробега
  - цены
- Адаптивный дизайн (Bootstrap)
- Hover-эффекты и современный UI

---

## 🛠️ Технологии

- Python 3
- FastAPI
- Jinja2 (шаблоны)
- Bootstrap 5
- APScheduler
- JSON (вместо БД)

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Установить зависимости 

```bash
pip install -r requirements.txt
```

### 3. Запустить сервер

```bash
uvicorn app.main:app --reload
```

### 4. Открыть в браузере

http://127.0.0.1:8000


## 💾 Хранение данных

Данные сохраняются в файл:

data/cars.json

Каждое обновление:

полностью перезаписывает файл
сохраняет только актуальные данные
исключает дубликаты
